# Isolated Relay Module

This project is a compact and reliable relay module designed to control high-voltage loads using a microcontroller (e.g., Arduino, ESP32). It features optical isolation to ensure the safety of your controller by separating the low-voltage control logic from high-voltage circuits.

## Features
- **Optical Isolation:** Prevents electrical noise and high-voltage spikes from damaging your microcontroller.
- **Compact PCB Design:** Optimized for space-constrained industrial or hobbyist applications.
- **Professional Documentation:** Includes an Interactive BOM for easy assembly.

## Project Contents
- `gucmodulu.kicad_pcb`: PCB layout file.
- `gucmodulu.kicad_sch`: Schematic design.
- `ibom.html`: Interactive Bill of Materials for assembly (open this in your browser).
- `gucmodulu.step`: 3D model of the module for mechanical integration.

## How to View
1. **To inspect the design:** You can open the project files using [KiCad](https://www.kicad.org/).
2. **For assembly:** Open the `ibom.html` file in your web browser. It provides a visual guide and a searchable list of components.
3. **3D Model:** You can view the module in 3D directly within the GitHub interface by opening the `gucmodulu.step` file.
